# ChatBot_Terapeutico
Projeto focado na criação de um chatbot terapeutico. Ainda está em andamento, nos passos iniciais, ainda sendo necessário um front end mais elaborado, um backend mais robusto também. Atualemente apenas a api da Groq, llama 3 com o chat para a IA está funcionando com um front mais basico, sendo possível conversar com a IA.
